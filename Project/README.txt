This is a Note syncing and editor web app in python flask framework. The notes are synced with user’s google drive. Users are asked to login with google and drive permissions are taken. This app makes a directory in user’s google drive account named “av_notes” and all notes are saved as files in that directory.
User can view the list of notes in av_notes directory after signing in with google, he/she can create a new note and view the notes alongwith their titles after he has created a note.

Pre-requisites for running this app:
Python 3.6.4
pip 9.0.1
Flask version: '0.12.2'
PyDrive
WTForms
Generated client_secrets.json file in the same directory
